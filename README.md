# EcoGuard Frontend

The interactive user interface for the EcoGuard environmental monitoring platform.

## ✨ Features
- **Global Monitoring Map**: Interactive Leaflet-based map for real-time deforestation tracking.
- **Reforestation Planner**: AI-powered interface for generating and viewing reforestation strategies.
- **Impact Dashboard**: Visualizing climate data and reforestation progress with 3D graphics.
- **Satellite Timeline**: Comparing historical and current satellite imagery.

## 🛠️ Technology Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **Animations**: GSAP (GreenSock)
- **Maps**: React Leaflet
- **3D Graphics**: React Three Fiber / Drei / Three.js
- **UI Components**: Lucide Icons, React CountUp, Swiper

## ⚙️ Setup
1. `npm install`
2. Configure `.env` with `VITE_API_BASE_URL`
3. `npm run dev` to start the development server.
